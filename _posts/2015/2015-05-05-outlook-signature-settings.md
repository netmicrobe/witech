1. 工具 / 选项... / 邮件格式 / 签名
1. 编辑签名,按Ctrl + F9 出现 域 { }  
2. 在此域中输入 { DATE @ "yyyy-MM-dd" } 保存编辑  
3. 在电子邮件中插入签名,按Ctrl + A 全选, 按F9 日期就自动出来了.  


### 日期/时间的格式

按CTRL + F9插入域，编辑位置出现{ }，在其中输入指令，例如

*  DATE @ "yyyy-MM-dd"
*  TIME @ "hh:mm:ss"
